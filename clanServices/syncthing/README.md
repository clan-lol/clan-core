## Usage

```nix
{
      instances.syncthing = {
        roles.peer.tags.all = { };
        roles.peer.settings.folders = {
          documents = {
            path = "~/syncthing/documents";
          };
        };
      };
}
```

Now the folder `~/syncthing/documents` will be shared and kept in sync with all your machines.
